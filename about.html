example metadata.json files

basic example: 
{
    "name": "Book 1",
    "sortname": "book 1",
    "author": "Author 1",
    "description": "This is an example book by author Author 1, who has sold over 1,000,000 copies of..."
}
examples for sortname:
sortname is used for sorting the list. if sortname is not defined it will put the first letter as 「ひ」 instead of "h", and then it will be at the bottom of the list. 
{
    "name": "ひだまりスケッチ",
    "sortname": "hidamari sketch",
    "author": "蒼樹うめ",
    "description": "(description)"
}

if all your stuff is in japanese you can make sortname as hiragana. there is no other reason to this other than, ① having the japanese character show up in the popup, and/or ② separating japanese and roman characters on purpose. 
{
    "name": "ひだまりスケッチ",
    "sortname": "ひだまりすけっち",
    "author": "蒼樹うめ",
    "description": "(description)"
}
